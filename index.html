Bootcamp Experience

Enrolling in Coding Dojo and progressing through the program was probably the most difficult yet fullfilling thing I have done to date. I came into this program knowing little to no knowledge of coding. All I had for experience was some knowledge of Javascript because of Codeacademy, and a little HTML knowledge because of the old Myspace days where everyone was customizing their profiles. How Coding Dojo works is that they teach you three full stacks.The first first two weeks however you do the web fundamentals track, which is focused on HTML, CSS, and front-end javascript libraries(ie. JQuery). Once that is completed, you move on to your first stack, Python. Python is a mandatory stack because it is a simple language to learn and has a very easy framework to work with. Typically each stack is one month long with that last week being a project week where you can work on a project for your portfolio! After Python you can choose whatever language they have available for your next two stack. I personally chose Java and the MEAN stack. For more information about my experiences with each stack check below!

  • Web Fundamentals
    Web fudamentals looking back now was by far the easiest track, but at the time I struggled with it a lot because of all new the concepts being thrown at me. During this time we also were taught how to use the terminal/gitbash, git, and github. I never knew how powerful the terminal was an how useful it is! What a great tool. Anyways, on my first day I was extremely nervous because of the fact that I had no college degree while all my cohort members that I talked to did plus work experience. The one advice I give to anyone that is thinking to go to a bootcamp with the same experience as me is make sure that programming is truly what you want to do! During my time at the bootcamp I've seen many people fall behind to the point of no return because they didn't have the determination to struggle through it. You have to have passion to complete this! It doesn't matter where your skill level is when you start, as long as you put in the work you will see the results, but you have to be willing to sacrafice! As the days went by, we started figuring out who were the strong coders in the group. Those people got asked many questions all the time and were always a step ahead of everyone else. I was not one of these people, I was a little above the middle of the pack when it came to coding strength. I didn't need to ask much questions, but I wasn't ahead of the curve either. Everyday we were taught a lot, we did a lot of HTML assignments from basic layouts, then to adding CSS to them, all the way to finally adding JQuery as well. In the end most of us passed our exam and we moved on to our Python track virtually unscathed.

  • Python

    Python was by far my favorite stack because I had such an awesome instructor. Fair warning to those thinking about going here, the instructors here are not the greatest. Lectures typically consist of what is on the learning platform already. Python is a really fun language and I highly suggest anyone who is trying to learn to code to learn this language first! The syntax is nice and simple and is very easy to read. Getting back on subject, the Python stack introduced us on how to make webpages and introduced us to GET and POST requests. Learning this at first was tough for me. I fell from being a middle of the pack student to the lower ends. I started spending late nights at the bootcamp just so I can catch up. Luckily as I stated before, I had a really great instructor who took the time to teach us and teach us right. A lot of people at this point began falling behind to point of no return. Many complained, some quit. To those who are in a bootcamp and a falling behind my advice is friend someone who is smarter than you in the bootcamp. If you can't friend someone know that you need to learn to make friends and communicate because if you don't have these technical skills you'll never get a job. Through this stack we learned how to make webpages, Flask, Django, MySql, and Sqlite. Big shoutout to my instructor Kevin. Kevin if you're reading this, thank you for setting me up for success.

    My python cohort!

  • Java

    OH JAVA. How I hate you and love you so. If I can redo anything about my experience it would be not learning this stack. By all means I do not hate java, it was the fact that my cohort was the first cohort to be introduced to the Java track. The learning platform for Java was horrible, and us students did not get much support from the instructors becuase only one instructor knew Java and he only finished this stack literally a month beefore us. In my opinion this was very unprofessional. As for the learning itself, I was very fortunate that I had a good instructor for python, because of the fundamentals I learned in Python I was able to understand the Java track. At this point I became in my opinion one of the strong coders in my cohort. I finished assignments on time and I was able to help those who needed help. It felt good to help other people! As for what we learned we basically did the same assignments in Python but this time in Java. Between Python and Java, Java can be very challenging. What can be done easily in Flask can take extremely long in Spring MVC. Even though I had a rough experience in Java, I still highly recommend people to learn Java or C#. I strongly believe that if you can code in one of those languages, you can code in any language.
